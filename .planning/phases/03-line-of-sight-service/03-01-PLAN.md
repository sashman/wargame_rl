<interfaces>
**Grid coords:** `WargameModel.location[0] == x`, `location[1] == y`; board spans `0 .. board_width-1`, `0 .. board_height-1`.

**Blocking:** `Callable[[int, int], bool]` — `True` means cell **blocks** LOS through it (interior only per CONTEXT D-10).

**Public APIs (names may vary slightly but behaviours locked):**
- `iter_los_cells(x0, y0, x1, y1, width, height) -> list[tuple[int, int]]` — inclusive segment along Bresenham; **empty list** if either endpoint OOB.
- `has_line_of_sight(x0, y0, x1, y1, width, height, is_blocking) -> bool` — `True` if every **interior** cell on that segment is non-blocking; **False** if OOB.
</interfaces>
